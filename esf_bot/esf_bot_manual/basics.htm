Basics


To start a game.
Start up Earth's Special Forces Beta 1.2.1, goto Multiplayer Game, LAN Game, Start Game.

Since version 3.1, the bot comes with 2 menu's:

 

Menu 1: The Regular Botmenu

Press equals (=), to bring up this menu.
As you can see in the picture, the botmenu is divided
into catagories.

Press the appropriate number on you keyboard to go
to the appropriate menu.
If you want to get started right away, you only have to
know the first 2 two buttons.

For example, if you want to add a bot to picks a random team, just press 1.
If you want a Goku bot on team Good, you press 2, a new menu will appear, featuring the skill of the bot, since we
just want to add a bot and worry about the skill later, we press 6, default skill.
Now another menu will appear, featuring the team selection, we want the bot to be on team good, press 1.
And yet another menu appears, this one features the class of the bot, press 2 to pick Goku.

Congratulations! a Goku bot appeared in team Good.

You are now able to add a bot to your game, so technically you can stop reading and start playing.

 

 


Menu 2 : The CommandMenu


This menu is more user friendly then the regular botmenu. It contains a few more options and you can use
your mouse to make an option. To activate this menu, just press the shift button.

This menu features a more simple and easy interface,
for most users this menu is recommended.

You can either press the appropriate number in front of
the option, or move your mousecursor over the option.

You are done!
You now know enough to use the bots properly.
