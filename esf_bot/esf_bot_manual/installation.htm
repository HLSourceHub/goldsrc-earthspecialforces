Installation

What do I need?
Earth's Special Forces 1.2, this version of ESF bot is not compatible with previous versions.

How to Install?
-Executable
Run the executable and install in the directory where you installed Earth's Special Forces beta 1.2.

-Zip
Unzip the contents of the zip file into your Earth's Special Forces directory.
(make sure you preserve folders)

After installation, you will find a new directory inside Earth's special Forces directory.
Esf_bot, these files are located inside this directory:

- bot.cfg
Here are some settings for the bot when you startup an game.
- skill.cfg
This file houses the various skill settings of the bots, such a s reaction speed.
- ingame_help
Fearures the help that is presented ingame.
- BotEdit.exe
This file is a program that allows you to edit the various bot personalities.
- bot_personalities.cfg
Contains the various bot personalities. (can be edited with "BotEdit.exe")
- SBotEdit.exe
This file can be used to edit the SuperBot settings.
- sbot.cfg
Contains the settings for the super bots, starting powerlevel, regeneration, transparancy, etc. (can be edited with "SBotEdit.exe")
- esf_bot_chat.txt
This file is for all the chat messages the bot uses, in the file are the instructions how the chat work and how you can alter some things.
- bot_names.txt
In this file all the names are listed wich the bot uses.
You can edit/add/remove them, can contain up to a maximum of 100 names.
- Activatebots.exe
This program will activate and deactivate the bots.
- \ esf_bot \ waypoints \ esf_bot_waypoint.txt
Some documentation for how you can create waypoints.(redundant)
- \ esf_bot \ esf_bot.dll
This is the main file for the bot. Without it it won't work.

Note: when running a server with the bots it is advisable to look inside these files, since most of them provide additional information.
